# Truth tables

## Solution 😎

1. T `&` T = T ✅
2. T `&` F = F ✅
3. F `&` T = T ❌
4. F `&` F = F ✅
5. T `|` T = T ✅
6. T `|` F = F ❌
7. F `|` T = T ✅
8. F `|` F = F ✅
9. `~`T = T ❌
10. `~`F = T ✅
11. (T `&` F) `|` (`~`F) = T ✅
12. (T `|` F ) `&` (F `|` F) = T ❌
13. `~`((T `|` F ) `&` (F `|` F)) `&` F = F ✅
14. `~`((T `|` F ) `&` (F `|` F)) `&` T = T ✅
